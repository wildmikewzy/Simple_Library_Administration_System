# 简易图书管理系统

一个基于C++开发的简易图书管理系统，支持图书查询、借阅、归还、浏览和管理功能。

## 功能特性

### 📚 核心功能
- **图书查询** - 按书名查询图书信息
- **图书借阅** - 借阅图书并更新库存
- **图书归还** - 归还图书并恢复库存
- **图书浏览** - 查看所有图书信息
- **图书管理** - 管理员添加图书和修改库存

### 🔐 安全特性
- 管理员权限验证（口令：`admin123`）
- 数据持久化存储
- 输入验证和错误处理

## 系统要求

- **操作系统**: Windows/Linux/macOS
- **编译器**: 支持C++11标准的编译器（如g++, clang++, MSVC）
- **依赖**: 标准C++库

## 快速开始

### 1. 编译程序
```bash
g++ -o library_management code.cpp
```

### 2. 运行程序
```bash
./library_management
```

### 3. 使用系统
程序启动后，您将看到主菜单.

## 使用说明

### 🔍 查询图书 (s/S)
- 输入完整书名（包括书名号）
- 系统显示图书详细信息（书名、作者、在馆数量）

### 📖 借阅图书 (b/B)
- 输入要借阅的图书名
- 系统检查库存并完成借阅
- 库存不足时会提示"该书全部借出"

### 📚 归还图书 (r/R)
- 输入要归还的图书名
- 系统自动更新库存数量

### 📋 浏览图书 (l/L)
- 显示所有图书的格式化列表
- 包含书名、作者、在馆数量信息

### ⚙️ 管理功能 (m/M)
- 需要输入管理员口令：`admin123`
- 支持添加新图书或增加现有图书库存
- 有3次输入机会，错误将返回主菜单

## 数据存储

系统使用文本文件存储图书数据：
- **文件路径**: `Library Imformation database.txt`
- **格式**: CSV格式，包含序号、书名、作者、在馆数量
